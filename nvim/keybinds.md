# Important keybindings
Ranked in order of importance


## Telescope
_ + _ => search files already open
_ + sf => search ALL files starting from start dir
_ + / => search inside current buffer
_ + sw => search for current word in other files
_ + sd => search diagnostics
_ + sc => search config files

## LSP
grd => Goto Definition (via Telescope)
    CTRL + T => go back
grr => Goto References
grn => Rename Current file
_ + th => toggle inlay hints


## Misc
Add/delete/replace surroundings (brackets, quotes, etc.)
 - saiw) - [S]urround [A]dd [I]nner [W]ord [)]Paren
 - sd'   - [S]urround [D]elete [']quotes
 - sr)'  - [S]urround [R]eplace [)] [']

# Flash
zk => Jumping to a letter

# Tabout
Tab => Get out of pairs forwards
Shift Tab => Get out of pairs backwards

# Comments
gc => Comment/Uncomment Selection
gcc => Comment/Uncomment Current Line
